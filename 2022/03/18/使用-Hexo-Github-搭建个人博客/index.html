使用 Hexo + Github 搭建个人博客

2022-03-18 286 words
hexo



1、所需要准备的环境

  • git-bash
  • nodeJs (npm)
  • vscode
  • github account



2、创建 github 仓库


仓库的名称必须是： username.github.io

访问 url ：https://www.username.guthub.io



3、配置 SSH key


测试是否配置成功：

ssh -T git@github.com



4、配置 hexo


第一步是安装：

npm install -g hexo

第二步时初始化：

cd ../hexo
hexo init

生成和启动服务：

hexo g
hexo s



5、hexo 主题


https://hexo.io/themes 下载相关的主题，并安装

安装到 hexo/themes 下


修改配置文件 _config.yml

theme: xxx

配置关联 github 信息：

deploy:
  type: git
  repository: git@github.com:username/username.github.io.git
  branch: master

安装插件并上传到 github：

npm install hexo-devoloper-git --save
hexo d

使用了 kaze 主题：
https://github.com/theme-kaze/hexo-theme-Kaze



6、hexo 常用指令


hexo n 'text'       # 新建文章
hexo n page 'text'  # 新建页面
hexo clean          # 清空配置
hexo g              # 生成页面
hexo s              # 开启端口映射
hexo d              # 上传代码
hexo help
hexo version

另外：

使用 <!-- more --> 可以控制文章显示的字数



7、github 上的设置


https://docs.github.com/cn/pages/quickstart

几个要求：

  • 仓库必须是 public
  • 随便选择一个主题
  • 等待30分钟
