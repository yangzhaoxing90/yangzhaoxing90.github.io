4. HTML Attribute

-- url link

<img src="img1.png" width="500" height="600"> 
-- image source

<p style="color:red;">This is a phragraph.</p>

6. HTML Styles


<body style="background-color:powderblue;"></body>
-- background color

<h1 style="color:blue">This is a heading</h1>

-- text font

-- text size
