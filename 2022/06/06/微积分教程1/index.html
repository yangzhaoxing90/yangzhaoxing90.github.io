有理数域

  1. 假设有理数已定义 (补充一些漏掉的定理定义)
    • 有理数等式两边同时加同个数, 等式仍然成立
    • 对于有理数 aaa, 有理数 −a-a−a 是有定义的
    • 对于有理数 aaa, 有理数 1a\dfrac 1 aa1​ 是有定义的, aaa 不等于 0
  2. 2\sqrt22​ 不存在的证明 -> 并不存在有理数 pq\dfrac p qqp​ 其平方能等于 2
  3. 有理数域的基本性质 (有理数用 aaa, bbb, ccc … 表示)
    • 有理数的序
      • a=ba = ba=b, a>ba > ba>b, a<ba < ba<b, aaa 与 bbb 之间必有且仅有关系之一成立 (三歧性)
      • 由 a>ba > ba>b, b>cb > cb>c 推得 a>ca > ca>c (传递性)
      • 若 a>ba > ba>b, 则必能求得一数 ccc, 使得 a>ca > ca>c, 且 c>bc > bc>b (稠密性)
    • 有理数的加法及减法
      • a+b=b+aa + b = b + aa+b=b+a (加法交换)
      • (a+b)+c=a+(b+c)(a + b) + c = a + (b + c)(a+b)+c=a+(b+c) (加法结合)
      • a+0=aa + 0 = aa+0=a
      • 对于数 aaa 存在数 −a-a−a, 使得 a+(−a)=0a + (-a) = 0a+(−a)=0
      • 差的定义： c+b=ac + b = ac+b=a, 数 ccc 为数 aaa 及 bbb 的差, 证明 ccc 存在且唯一, 记 c=a−bc = a - bc=a−b
        • 求证 0=a−a0 = a - a0=a−a 唯一性
        • 求证 −a=0−a-a = 0 - a−a=0−a 唯一性
        • 求证 a+(−a)=(−a)+a=0a + (-a) = (-a) + a = 0a+(−a)=(−a)+a=0
        • 求证 a=−(−a)a = -(-a)a=−(−a)
        • 求证 −(a+b)=(−a)+(−b)-(a+b) = (-a) + (-b)−(a+b)=(−a)+(−b)
      • 由 a>ba > ba>b 推得 a+c>b+ca + c > b + ca+c>b+c
        • 求证 a>ba > ba>b, 则 −a<−b-a < -b−a<−b
        • 求证 a>0a > 0a>0, 则 −a<0-a < 0−a<0
      • 有理数的绝对值 ∣a∣=∣−a∣|a| = |-a|∣a∣=∣−a∣
      • 求证 a>ba > ba>b 及 c>dc > dc>d 推出 a+c>b+da + c > b + da+c>b+d
    • 有理数乘法及除法
      • ab=baab = baab=ba (乘法交换)
      • (ab)c=a(bc)(ab)c = a(bc)(ab)c=a(bc) (乘法结合)
      • a⋅1=aa \cdot 1 = aa⋅1=a
      • 对于异于 0 的数 aaa, 必有数 1a\dfrac 1 aa1​ (其倒数), 使 a⋅1a=1a \cdot \dfrac 1 a = 1a⋅a1​=1
      • 除的定义： c⋅b=ac \cdot b = ac⋅b=a, 则称 ccc 为 aaa 及 bbb 的商, 证明 ccc 存在且唯一, 记 c=abc = \dfrac a bc=ba​ 或 c=a:bc = a : bc=a:b
      • (a+b)⋅c=a⋅c+b⋅c(a + b) \cdot c = a \cdot c + b \cdot c(a+b)⋅c=a⋅c+b⋅c (乘法分配)
        • 求证 (a−b)⋅c=a⋅c−b⋅c(a - b) \cdot c = a \cdot c - b \cdot c(a−b)⋅c=a⋅c−b⋅c
        • 求证 b⋅0=0⋅b=0b \cdot 0 = 0 \cdot b = 0b⋅0=0⋅b=0
      • a>ba > ba>b 及 c>0c > 0c>0 推得 a⋅c>b⋅ca \cdot c > b \cdot ca⋅c>b⋅c
        • 求证 (−a)⋅b=−(a⋅b)(-a) \cdot b = -(a \cdot b)(−a)⋅b=−(a⋅b)
        • 求证, 若 a<0a < 0a<0, b>0b > 0b>0, a⋅b<0a \cdot b < 0a⋅b<0
        • 求证, 若 a>0a > 0a>0, b<0b < 0b<0, a⋅b<0a \cdot b < 0a⋅b<0
    • 求证有理数序的稠密性
    • 阿基米德公理
      • 不论 c>0c > 0c>0 是怎样的数, 总有大于 ccc 的自然数 nnn 存在着 (从几何的角度考虑)
      • 任意定长线段 A 和 B, A 重复若干次相加后, 定能超过 B

无理数的导入: 实数的域

  1. 无理数的定义
    • 分划 (把全体有理数拆分为两个非空集合 AAA, A′A'A′)
      • 任一有理数, 必在且仅在 AAA 及 A′A'A′ 二集之一中出现
      • 集 AAA 内的任一数 aaa, 必小于集 A′A'A′ 内的任一数 a′a'a′
      • 集 AAA 称为分划的下组, 集 A′A'A′ 为上组, 分划记作 A∣A′A|A'A∣A′
      • 验证 a<1a < 1a<1 和 a>=1a >= 1a>=1 形成分划
      • 分划的上下组只能有三种情况
        • 下组 AAA 内无最大数, 而在上组 A′A'A′ 内有最小数 rrr (分划由有理数 rrr 产生)
        • 下组 AAA 内有最大数 rrr, 而在上组 A′A'A′ 内无最小数 (分划由有理数 rrr 产生)
        • 在下组内既无最大数, 在上组内亦无最小数 (界数并不存在, 分划不由无理数产生)
    • 分划的第三种情况用于定义无理数, 把无理数 α\alphaα 理解为有理数域中确定它的分划 A∣A′A|A'A∣A′
    • 有理数由第一种情况定义 (约定)
  2. 实数的定义
    • 有理数和无理数总称为实数
  3. 实数域的序
    • 无理数的相等, 分划上组和下组分别重合; 反之, 若分划上下组分别重合, 则无理数相等; (有理数相等也可以兼容)
    • 无理数大于, 若 α\alphaα 为分划 A∣A′A|A'A∣A′, β\betaβ 为分划 B∣B′B|B'B∣B′, 若 AAA 包含 BBB, 则称 α\alphaα > β\betaβ (可以兼容有理数)
    • 实数性质证明
      • 任一实数 α\alphaα 与 β\betaβ 之间必有下列三种关系之一, α=β\alpha = \betaα=β, α>β\alpha > \betaα>β, α<β\alpha < \betaα<β (三歧性)
      • 由 α>β\alpha > \betaα>β, β>γ\beta > \gammaβ>γ 推得 α>γ\alpha > \gammaα>γ (传递性)
      • 两个实数 α\alphaα 及 β\betaβ, 其中 α\alphaα > β\betaβ, 恒有一个位于它们中间的有理数 r:α>r>βr: \alpha > r > \betar:α>r>β (因此, 这种有理数有无穷个) (稠密性)
      • 给定两实数 α\alphaα 和 β\betaβ. 若任取一个数 e>0e > 0e>0, 数 α\alphaα 及 β\betaβ 都能位于同一对有理数 sss 与 s′s's′ 之间 s′>α>ss' > \alpha > ss′>α>s, s′>β>ss' > \beta > ss′>β>s, 这对数的差小于 eee, s′−s<es' - s < es′−s<e, 则数 α\alphaα 与 β\betaβ 必须相等 (考虑如何证明)
